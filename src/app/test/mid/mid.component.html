    理想： 在这个前端混乱的时代，做项目最好是兼容各种技术，各种浏览器，组件化开发，模块化开发，快速开发，还要方便自动化测试。

    1. 集成 jQuary，项目中可以写 jQuary 代码也可以写 Angular 代码。


    2. 支持 TypeScript，项目中可以写 TypeScript 也可以写 JavaScript 代码。


    3. 支持SCSS，项目中可以写CSS样式也可以写SCSS代码。


    4. 集成阿里开源组件库，项目中可以使用 BootStarp 布局样式，也可以使用 nz-zrror 布局样式， 你可以增加其他开源组件库，说明文档中有个集成“饿了么”团队开源库的示例。
